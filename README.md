# DreamIT PPT Generator

AI 기반 프레젠테이션 자동 생성 서비스 - OpenAI/Claude를 활용한 전문 슬라이드 제작

## 사이트 정보

- **URL**: https://ppt.dreamitbiz.com
- **기술 스택**: React 19 + Vite + TypeScript + Supabase
- **AI 엔진**: OpenAI GPT-4o / Claude Sonnet
- **내보내기**: HTML-ZIP, PDF, PPTX

## 주요 기능

- AI 기반 프레젠테이션 자동 생성 (주제 입력 → 슬라이드 생성)
- 8가지 슬라이드 템플릿 (표지, 목차, 섹션표지, 본문, 다이어그램, 워크북, 요약, 뒤표지)
- 8가지 색상 테마 프리셋
- 세로(A4) / 가로(16:9) 방향 선택
- 인라인 텍스트 편집
- 3종 내보내기 (HTML-ZIP, PDF, PPTX)
- 유료 구독 (Free / Basic / Pro)
