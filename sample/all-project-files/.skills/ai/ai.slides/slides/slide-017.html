학습 목표

유용한 대화를 다시 찾을 수 있도록 정리하고, 필요하면 동료와 공유한다.

시나리오
Step 02~03을 거쳐 유용한 회의 안건 정리 + OJT 일정표를 만들었습니다. 이건 매주, 매학기 반복적으로 쓰게 될 패턴입니다. 다음 주에 같은 작업을 할 때 이 대화를 다시 꺼내 참고하고 싶습니다.

따라하기

1) 대화 제목 바꾸기 - 좌측 사이드바 대화 목록에 마우스를 올리고 ✏️(또는 ⋮) 아이콘을 클릭하여 의미 있는 이름으로 변경합니다.
2) 이모지·태그 활용 - 제목 앞에 📌, ✅ 같은 이모지를 붙이거나 [회의], [OJT] 등의 태그를 사용하여 직관적으로 구분합니다.
3) 공유 링크 생성 - 유용한 패턴은 팀 내 표준 템플릿으로 공유합니다. (※ 교내 데이터 공유 권한 정책 확인 필요)

원래 제목 : '내일 부서 정기 회의에서 다룰...'

바꾼 제목 : '📅 정기회의 안건 정리 템플릿'

직접 해보기 — 본인의 대화 3개 제목을 의미 있게 정리해 보세요

회의 관련 대화
보고서 관련 대화
데이터 분석 관련 대화

TIP 제목 규칙(이모지 + 업무키워드 + 날짜)을 일관되게 쓰면 한 달 후에도 빠르게 찾을 수 있습니다.
예: '📅 [회의] 2026-04 정기회의 템플릿'
