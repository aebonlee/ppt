이미지·PDF
자료 다루기

학습 목표이미지·PDF 같은 비정형 자료를 멀티모달 모델로 분석·요약한다.

시나리오: 50페이지짜리 외부 정책 보고서 PDF에서 본인 부서와 관련된 부분만 빠르게 파악해야 합니다.

따라하기

1) 멀티모달 모델 선택 - GPT-4o 또는 Gemini 1.5 Pro (이미지/PDF 인식 가능)
2) 파일 첨부 후 프롬프트 - "이 PDF에서 [학사행정] 관련 내용만 5줄로 요약. 출처 페이지 번호 함께"
3) 결과 검증 - 모델이 요약한 내용을 원본에서 실제 페이지 확인

활용 예시

자료 형태

활용 방법

이미지

회의 화이트보드 사진 → 회의록 텍스트 변환

PDF

정책 보고서 → 핵심 요약 + 페이지 인용

스캔 문서

손글씨 메모 → 디지털 텍스트 변환
