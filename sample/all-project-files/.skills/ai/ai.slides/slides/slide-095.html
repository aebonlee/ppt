인사 평가 코멘트 다듬기

1/2

학습 목표

부서장이 작성한 거친 메모 형태의 평가 노트를 격식 있고 왜곡 없는 평가 코멘트(강점·개선·종합)로 다듬는다.

시나리오

부서장이 5명의 직원 평가 노트를 두서없이 메모로 적어 두었습니다. 형식은 「대리 B — 자료 빠르게 만듦, 근데 가끔 디테일 놓침. 팀 협력은 나쁘지 않음.」 식입니다. 이 메모를 인사 평가 시스템에 들어갈 격식 있는 평가 코멘트로 변환해야 합니다.

따라하기 — 평가 코멘트 다듬기 프롬프트

[역할] 너는 학교 인사 평가 코멘트 다듬기 전문가다. [작업] 다음 거친 메모를 평가 시스템에 들어갈 격식 코멘트로 변환해 줘. [입력 메모] 대리 B — 자료 빠르게 만듦, 근데 가끔 디테일 놓침. 팀 협력은 나쁘지 않음. [출력 형식] == 대리 B == [강점] - (메모의 긍정 요소를 격식 있는 한 문장으로) [개선 영역] - (메모의 부정 요소를 비판이 아닌 「개선 영역」 어조로) [종합] - 1~2 문장으로 균형 있게 정리 [원칙 — 절대 위반 금지] 1. 메모에 없는 사실 추가 금지 (예: 메모에 「창의성」 언급 없으면 만들지 말 것) 2. 메모의 강도 임의 변경 금지 (예: 「가끔」 → 「자주」 변경 금지) 3. 차별 표현 금지 (성별·연령·외모·결혼) 4. 평가가 모호한 경우 「관찰 빈도가 적어 추가 관찰 필요」로 표기 5. 본인이 들어도 수긍할 만한 톤 — 비판이 아닌 객관 진술 6. 「업무 중 ~한 모습 관찰됨」 형식의 객관 사실 위주 [추가 출력] - 마지막에 「변환 점검」 섹션: ✅ 메모 외 내용 추가: 0건 ✅ 강도 변경: 0건 ✅ 차별 표현: 0건
