문서·인사 행정 자동화

직원 교육
일정·커리큘럼

1/2

학습 목표

신규 임용된 일반 임기제 직원 1명을 위한 5일 OJT 일정과 점진적 책임 부여 중심의 커리큘럼을 자동 생성한다.

시나리오

학사지원과에 신규 직원 1명이 다음 주 월요일에 출근합니다. 첫 5일 동안 부서 적응과 핵심 업무 학습을 위한 OJT 일정·커리큘럼이 필요합니다. 멘토는 대리 B(3.5년 경력)입니다.

따라하기 — 5일 OJT 설계 프롬프트

[역할] 너는 신규 임용 직원의 OJT 커리큘럼 설계자다.

[작업]
다음 정보를 바탕으로 5일 OJT 일정·커리큘럼을 만들어 줘.

[정보]
- 부서: 학사지원과
- 신규 직원: 일반 임기제, 학사 데이터 관리 직무
- 멘토: 대리 B (경력 3.5년, 같은 직무)
- 신규 직원의 사전 경험: 학사 행정 무경험, 엑셀·한글 사용 가능
- 가장 시급한 업무: 학적 변동 데이터 입력·검증 (출근 2주 차부터 일부 담당)

[형식 — 5일 OJT]
일자별 / 시간대별 / 활동 / 담당
- 1일 차: 부서 소개·환영·기본 시스템 계정 발급
- 2일 차: 핵심 업무 관찰 (멘토 따라다니기)
- 3일 차: 실습 시작 (멘토 옆에서 함께 작업)
- 4일 차: 실습 확대 (간단 업무 단독 수행)
- 5일 차: 평가·피드백·다음 주 계획

[설계 원칙 — 점진적 책임 부여]
1. 1~2일 차: 「관찰만」 — 책임 부여 없음
2. 3일 차: 「함께 작업」 — 멘토가 마지막 검토
3. 4일 차: 「자체 작업 + 멘토 즉시 도움」
4. 5일 차: 평가는 「학습 진행도」 중심, 결과 평가가 아님

[안전 장치]
- 학생 개인정보가 들어간 실제 데이터는 4일 차까지 사용 금지
- 1.6절 보안 윤리 교육은 1일 차에 반드시 포함
- 일정에 점심·휴식 시간 포함 (09:00~18:00, 점심 12:00~13:00)

[출력]
일자별 시간표 표 + 일자별 학습 목표 1줄 + 종합 의견 3줄
