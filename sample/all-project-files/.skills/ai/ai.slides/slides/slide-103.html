== 학사 데이터 관리 인수인계서 ==

1. 직무 개요
이 자리는 학적 변동·통계·공문을 매주 처리합니다. 외부 자료 응답이 잦아 시간 관리가 핵심입니다.

2. 주 단위 업무 흐름
월: 신규 학적 변동 처리 (AM) / 통계 정리 (PM)
화: 부서 회의 + 회의록
수: 공문 발송, 외부 응답
목: 통계 검증, 다음 주 준비
금: 주간 보고서

3. 핵심 협력 동료
- 대리 B (학사지원과, 멘토): 학적 절차 모를 때 즉시 문의
- 주임 C (인사과): 인사 발령 관련 데이터 교환
- 주임 D (정보화본부): 시스템 권한·오류 시 연락

4. 자주 발생하는 어려움
- 매월 1주차 월요일: 학적 변동 3배 → 토요일에 미리 검토
- 학기 시작 2주 전: 야근 발생 → 사전 일정 비워두기
- 졸업 사정 (12·6월): 자료 검증 시간 2배

후임자 첫 주 가이드 — 5일

권장 행동

1일 차

본인(전임자)과 1시간 동행, 업무 시스템 계정 발급

2일 차

학적 변동 처리 흐름 관찰 (혼자 처리는 X)

4일 차

단독 처리 1건 + 즉시 검토

5일 차

1주 회고 + 다음 주 계획 + 본인은 인사과로 이동
