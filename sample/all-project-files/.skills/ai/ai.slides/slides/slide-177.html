차트 텍스트
자동 작성 1/2

학습 목표

차트의 제목·축 라벨·단위·출처·캡션 등 6가지 텍스트 요소를 부서장 보고용 격식 있는 톤으로 자동 작성한다.

시나리오

부서별 출장비 묶은 막대 차트를 부서장 보고용으로 마무리해야 합니다. 차트 그림은 완성됐지만, 아직 제목, 축 라벨, 단위, 출처, 그리고 캡션이 명확하게 정리되지 않았습니다.

따라하기 프롬프트

역할: 차트 텍스트 작성 비서

작업: 차트 6가지 텍스트 요소 표준화

차트 정보: 묶은 막대 차트, X축=월, Y축=출장비(원), 시리즈=학사지원과/학생지원처/정보화본부

산출물:
1. 제목
2. 부제
3. X축 라벨
4. Y축 라벨+단위
5. 출처(데이터 기준일)
6. 캡션 1줄

톤: 부서장 보고용 격식 있는 톤

형식: 6개 항목 표

안전장치: 추정한 단위·기간은 반드시 [확인 요망] 표기
