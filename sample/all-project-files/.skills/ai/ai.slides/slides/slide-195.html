학습 목표

예산 전용 또는 추경 사유서를 표준 공문 양식으로 자동 작성한다.

시나리오

도서비 잔여액(2,000,000원)을 외부강사료로 전용하기 위한 사유서를 작성합니다. 학기 중 예상치 못한 외부 강사 초청 결정으로 도서비 일부의 전용이 필요합니다.

따라하기 프롬프트

역할: 예산 전용·추경 사유서 작성 비서

작업: 표준 공문 양식 사유서 작성

전용 정보:
• 전용 항목: 도서비 → 외부강사료
• 전용 금액: 2,000,000원
• 전용 사유: 학기 중 예상치 못한 외부 강사 초청 결정
• 전용 후 사용 계획: 외부 특강 2회 진행

양식: 수신·제목·근거·전용 사유·전용 내용 표·붙임·결재 라인 포함

형식: 학교 공문 표준 양식 사용
