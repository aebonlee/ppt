학습 목표

매월 말 반복되는 마감 작업 7단계를 LLM 카드 운용으로 자동화하여 작업 시간을 30분 이내로 압축한다.

시나리오

매월 말 학사지원과의 정기 마감을 진행해야 합니다. 반복되는 확인 작업들을 7가지로 분류하여 자동화 카드를 활용합니다. (①출근부 점검, ②예산 집행률, ③영수증 처리, ④이상 거래 점검, ⑤회의록 통합, ⑥민원 통계, ⑦보고 자료 작성)

7단계 워크플로우 구조

① 카드 호출 : 미리 작성해 둔 해당 업무 프롬프트 카드를 불러옵니다.

② 변수 치환 : 이번 달에 해당하는 기초 데이터(월, 부서명 등)를 변경합니다.

③ 입력 : 정리된 프롬프트를 전남대GPT에 복사 및 붙여넣기 합니다.

④ 결과 수령 : 생성된 결과를 복사하여 문서나 엑셀에 가져옵니다.

⑤ 본인 검증 : 결과물의 이상 여부(환각 확인)를 반드시 검토합니다.

⑥ 통과/수정 : 검증 완료 시 확정, 오류 시 프롬프트를 재작성합니다.

⑦ 통합 : 7가지 단계별 결과를 하나의 파일로 통합하여 보고 자료로 만듭니다.
