전남대GPT 단축키

1. 전남대GPT 주요 단축키

단축키 기능 설명
Ctrl + Enter 메시지 전송 입력 후 즉시 전송
Shift + Enter 줄바꿈 같은 메시지 내 새 줄
Ctrl + / 명령 팔레트 메뉴 검색
Ctrl + K 새 대화 새 채팅 시작
Ctrl + Shift + S 대화 저장 현재 대화 즐겨찾기
Esc 입력 취소 응답 생성 중단
Ctrl + B 사이드바 대화 목록 토글

2. 효율 팁 5가지

① 자주 쓰는 프롬프트는 즐겨찾기

② 모델 변경은 같은 대화 내에서 가능

③ 긴 문서는 분할해서 입력

④ 코드는 마크다운 ``` 사용

⑤ 결과 복사는 'Copy as Markdown' 활용

💡 단축키 표를 인쇄해 모니터 옆에 부착해 두시면 업무 효율이 크게 향상됩니다.
