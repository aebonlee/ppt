실습 환경 준비 · 계정 점검
ChatGPT와 Microsoft 365 Copilot 계정을 5분 안에 점검합니다.
학습 목표
회사 계정으로 ChatGPT(또는 Enterprise)와 Microsoft 365 Copilot에 로그인하고, 데이터 학습 차단 옵션이 켜져 있는지 확인합니다.
시나리오
월요일 아침, 부서장이 "다음 주부터 견적은 ChatGPT로 만들어 보자"고 합니다. 먼저 계정과 보안 옵션부터 확인해야 합니다.
따라하기 — 4단계
1)  회사 계정 확인
개인 Gmail이 아닌 회사 메일(@company.co.kr)로 가입했는지 확인 — 라이선스 관리·감사 추적이 가능해진다.
2)  ChatGPT 데이터 학습 차단
설정 → 데이터 컨트롤 → Improve the model for everyone 토글 OFF.
3)  Microsoft 365 Copilot 라이선스
Word · Excel · Outlook 우측 상단 Copilot 아이콘이 보이는지 확인 — 안 보이면 IT 부서에 라이선스 요청.
4)  첫 대화 만들기
"안녕, 너는 누구야?"를 보내고 응답이 오면 환경 준비 완료.

주의 · CAUTION
개인 계정으로 회사 견적·고객명·차대번호를 입력하면 모델 학습 데이터로 흘러갈 수 있습니다. 반드시 회사 계정 + 학습 차단 OFF 조합을 확인한 뒤 본 교재의 실습을 진행하세요.
