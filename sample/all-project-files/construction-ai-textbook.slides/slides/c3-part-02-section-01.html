2. 30초 결정 — 두 줄 룰
RULE 01
입력이 50쪽 이상 → Claude
매뉴얼·계약서·긴 회의록을 통째로 넣을 일이라면 무조건 Claude. 토큰 한도와 인용 정확도가 ChatGPT를 앞섭니다.
RULE 02
최신 정보 필요 → Genspark
"이번 달 시장 동향", "경쟁사 신제품" 같은 시점성 질의는 Genspark가 1순위. 모델 학습 컷오프 너머의 정보를 직접 조사합니다.
KEY POINT
"한 도구 = 한 업무" 가 아닙니다. 위의 5가지 시나리오 중 자기 부서 업무 1개를 골라, 1순위 + 2순위 도구를 책상 앞에 적어 두세요. 결정 피로가 사라집니다.
⁴ 시나리오 매핑은 건설직무 표준 업무 5종(견적·정비·매뉴얼·보고·조사)을 기준으로 구성 043
