How to use this book

1. 한 Step의 구성

모든 실습은 동일한 5개 블록으로 구성됩니다.

블록

무엇을 하는가

학습 목표 (회색 박스)

무엇을 할 수 있는지 한 줄 정리

시나리오 (본문 단락)

학교 업무에서 만날 법한 구체적 상황

핵심 프롬프트 (코드 박스)

그대로 복사해 붙여넣을 완성형 프롬프트

예상 결과 (베이지 박스)

입력 시 받게 될 결과 예시

직접 해보기 (점선 박스)

본인 업무로 변형해 시도

추가로 💡 TIP과 ⚠️ 주의 박스가 들어 있습니다.

2. 책을 읽는 순서

  • 가능하면 1장부터 순서대로
  • 1.1~1.6은 모든 장의 전제
  • AI 경험자는 1.1~1.2 Step 01~05만 빠르게, 본인 업무 챕터로 이동

3. 실습 환경

  • 브라우저: Chrome 또는 Edge 최신
  • 운영체제: Windows 10+ / macOS 12+
  • 계정: 전남대 메일(@jnu.ac.kr)
  • 추가 도구: MS Excel·Word (또는 한셀·한글)

4. 본문 기호

[대괄호]
본인 상황에 맞게 바꿔 넣을 부분
▶
클릭해야 할 메뉴·버튼
→
다음 화면 또는 다음 단계
▽
펼쳐서 봐야 할 추가 정보

5. 문제가 생겼을 때

  • 로그인 안 됨 → 학교 메일 비밀번호 만료 확인
  • 결과가 매번 다름 → 정상 (Step 03 참고)
  • 거짓 정보 → 반드시 검증 (1.6절 참고)

💡 책 자체의 오류나 개선 제안은 전남대학교 정보화본부로 보내주세요. 다음 개정판에 반영됩니다.
