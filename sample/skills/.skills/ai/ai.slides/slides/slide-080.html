슬라이드 텍스트

1/2

학습 목표

회의 자료나 보고서를 발표용 슬라이드 5장 분량의 글머리 기호 텍스트로 변환한다.

시나리오

다음 주 부서 워크숍에서 1분기 결과를 5분 동안 발표해야 합니다. 슬라이드는 5장 정도가 적절합니다. Step 10에서 만든 1쪽 요약 보고서를 슬라이드 텍스트로 변환합시다.

따라하기 — 프롬프트

[작업] 다음 1쪽 요약 보고서를 5분 발표용 슬라이드 5장 텍스트로 재구성해 주세요.

[슬라이드 5장 구성]
- 슬라이드 1: 제목과 발표 목적 (제목 1줄 + 부제 1줄)
- 슬라이드 2: 현황 (3개 글머리 기호, 각 1줄)
- 슬라이드 3: 핵심 수치 (3개 글머리 기호 또는 표 1개)
- 슬라이드 4: 결정 필요 사항 (2개 글머리 기호, 각 2줄까지)
- 슬라이드 5: 2분기 계획 + 마무리 멘트 (각 1줄)

[톤]
- 발표용이므로 짧고 명료
- 한 글머리 기호당 30자 이내
- 데이터 수치는 반드시 포함
- 청중 시선이 화면에 5초 이내 머물도록

[형식]
- 각 슬라이드 시작에 「===== 슬라이드 N · 제목 =====」 표시
- 글머리 기호는 「•」 사용
- 강조 부분은 **굵은 글씨**

[안전 장치]
- 보고서에 명시되지 않은 정보를 추가하지 마세요
- 한국어로 작성
