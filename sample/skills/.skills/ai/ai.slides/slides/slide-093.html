면접 질문지 설계

학습 목표

채용 직무에 맞는 STAR 기법 기반 면접 질문 8개와 5점 척도 평가 루브릭을 자동 생성한다. 면접관 간 평가 일관성을 높인다.

STAR 기법이란

S — Situation (상황): 어떤 상황이었는가?
T — Task (과제): 무엇을 해야 했는가?
A — Action (행동): 본인이 어떻게 행동했는가?
R — Result (결과): 그 결과는 어떠했는가?

따라하기 — 질문지 + 루브릭 프롬프트

[역할] 너는 대학 채용 면접 질문지 설계 전문가다. [입력] (Step 15에서 작성한 채용 공고문 붙여넣기) [작업] 공고문의 「주요 업무·자격 요건」을 기반으로 면접 질문 8개를 STAR 기법으로 설계해 줘. 각 질문에는 5점 척도 평가 루브릭을 함께 제공. [질문 구성 — 8개] 1) 직무 적합성 — 2개 (주요 업무 직접 경험) 2) 문제 해결 — 2개 (까다로운 상황 대응) 3) 협업·의사소통 — 2개 (팀워크·갈등) 4) 학습 의지 — 2개 (새 도구·환경 적응) [각 질문 출력 형식] 질문 N — [영역] [질문 본문 — STAR 형태로 답할 수 있게] 평가 루브릭 (5점 척도) - 5점: 명확한 사례 + 구체 수치 + 본인 주도성 - 4점: 명확한 사례 + 본인 행동 명시 - 3점: 사례는 있으나 본인 역할 모호 - 2점: 사례 부재 / 일반론 답변 - 1점: 답변 회피 또는 직무 무관 [원칙] - 차별 질문 금지 (가족·연령·결혼·종교·정치) - 직무와 연관 없는 사적 질문 금지 - 면접관마다 다른 질문 사용 금지
