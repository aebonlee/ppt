학습 목표

평균·표준편차·이상치를 자동으로 식별하고 1쪽 요약 보고서로 정리한다.

시나리오

강의 만족도 설문 결과 200건(5점 척도)이 수집되었습니다. 강좌별·문항별 평균을 분석하고 의미 있는 통계를 요약해야 합니다.

따라하기 프롬프트

[역할]

당신은 정확한 데이터 해석을 돕는 통계 분석 비서입니다.

[작업]

제공된 데이터를 분석하여 기초 통계 요약 보고서를 작성해 주세요.

[데이터]

강의 만족도 200건 5점 척도 데이터 (데이터 첨부)

[산출물 및 분석 기준]

• 1쪽 A4 분량으로 요약
• 전체 평균, 강좌별 평균 계산
• 이상치 식별 (기준: 평균에서 ±2σ 벗어난 값)

[형식]

마크다운 표 + 주요 결과 요약문

[안전장치]

응답 수 30건 미만인 강좌는 통계적으로 신뢰하기 어렵다는 점을 명확히 표기해 주세요.
